# SocialConnect - Social Media Platform

## วิธีติดตั้งและเริ่มใช้งาน

### ความต้องการ
- Node.js v18+ (ดาวน์โหลด: https://nodejs.org)

### ขั้นตอน

```bash
# 1. เข้าโฟลเดอร์โปรเจค
cd D:\project

# 2. ติดตั้ง packages
npm install

# 3. เริ่ม server
npm start
```

จากนั้นเปิด browser ไปที่: **http://localhost:3000**

## ฟีเจอร์ที่มี

### 🔐 ระบบบัญชี
- สมัครสมาชิก (ชื่อ, อีเมล, เบอร์โทร, อาชีพ, ที่อยู่, สัญชาติ, ภาษา)
- ยืนยันอีเมล + ตั้งรหัสผ่าน
- เข้าสู่ระบบ

### 🛡️ e-KYC
- แพ็กเกจรายเดือน 330 บาท / รายปี 2,000 บาท
- สแกนใบหน้า, บัตรประชาชน (หน้า/หลัง), Visa/Passport
- ผูกบัญชีคริปโต
- เครื่องหมาย ✓ (verified badge)

### 👤 Profile
- URL แบบกำหนดเอง
- แชร์โปรไฟล์
- ติดตาม/ยกเลิกติดตาม
- โพสต์ (ถูกใจ, คอมเม้นท์, แชร์, วิว, โดเนท)
- แชท
- ดูมาร์เก็ตเพลส (ซื้ออย่างเดียว)
- ดูไลฟ์สด (ดูอย่างเดียว)

### 🏪 Page
- สร้างเพจธุรกิจ
- URL แบบกำหนดเอง
- ลงประกาศขายในมาร์เก็ตเพลส ✓
- เริ่มไลฟ์สด ✓

### 💬 Chat, 🔔 แจ้งเตือน, ⚙️ ตั้งค่า

## โครงสร้างไฟล์

```

    ├── style.css
    └── app.js
```
